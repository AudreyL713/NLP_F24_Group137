Hello! And welcome to the data collection site for our 6.8610 (Natural Language Processing) final project. We are studying the effect of using keystroke timing data to supplement NLP password-cracking models (i.e. PassBERT). As you type out the prompts on this website, we will be collecting keystroke timing data. These will not be connected to your personal information in any way. Thanks in advance for your help!

Instructions:

  • You will be asked to type out 20 unique passwords, as if you are entering them to log in to a website.
  • You will be asked to type each password 3 times
  • When you are done typing a password, please press the "Enter" key to proceed.
  • If you make a mistake, we will ask you to try again and this will not count towards the total 3 times.
  • Even if you exit this page before completing all 20 passwords, the data will still be sent to us for the passwords that you did complete.
